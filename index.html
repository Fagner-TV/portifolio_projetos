O trabalho demonstra habilidades em modelagem de dados, criação de visualizações dinâmicas e storytelling com dados, fornecendo insights claros para suporte a decisões estratégicas.

Construção de análise de desempenho

Foi realizada uma análise de negócio em uma rede de cafeterias com o objetivo de avaliar a saúde do empreendimento, identificar os fatores que influenciam o bom ou mau desempenho e prever os resultados para os próximos 30 dias. A partir dos dados coletados, foram elaboradas recomendações estratégicas tanto para potencializar o crescimento quando o cenário é favorável, quanto para reverter situações de queda de desempenho, garantindo decisões orientadas a dados e maior alavancagem do negócio.
